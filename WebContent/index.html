Gestion des données de base

Les données de base se sont les données qui présentent les différentes parties qui constituent l'établissement tel que service , département , salle est laboratoire Bureaux. Le responsable de gestion des de base doit pouvoir gérer ces élément e lui donnant la possibilité d'ajouter supprimer, ou de modifier ces informations. L'initialisation de ces données de bases est la première étape avant de pouvoir manipuler les différents modules de l'application.
